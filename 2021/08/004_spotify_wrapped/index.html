My Spotify wrapped music App

August 20, 2021 in App, Solo project

An App that returns my top songs from Spotify in the manner of Spotify’s wrapped.

App that returns my 10 top songs over three periods of time (recent, past months, past year) - made following this John Manelli’s tutorial.

DATA

The data is retrieved from my Spotify account with the help of the Spotify API. The app is programmed to return the top ten songs.

TOOLS & TECHNIQUES

  • Spotify API
  • Googlesheet API
  • Glide (no code site to create an app)

Code is visible on Github.
